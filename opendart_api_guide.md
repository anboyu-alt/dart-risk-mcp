# OpenDART API 개발가이드 전체 정리

> 금융감독원 전자공시시스템(DART) Open API
> 출처: https://opendart.fss.or.kr/guide/main.do
> 정리일: 2026-04-22

---

## 공통 사항

- **Base URL**: `https://opendart.fss.or.kr`
- **Method**: GET
- **Encoding**: UTF-8
- **Output**: JSON (`.json`) / XML (`.xml`)

### 공통 에러코드

| 코드 | 메시지 |
|------|--------|
| `000` | 정상 |
| `010` | 등록되지 않은 키입니다. |
| `011` | 사용할 수 없는 키입니다. 오픈API에 등록되었으나, 일시적으로 사용 중지된 키를 통하여 검색하는 경우 발생합니다. |
| `012` | 접근할 수 없는 IP입니다. |
| `013` | 조회된 데이타가 없습니다. |
| `014` | 파일이 존재하지 않습니다. |
| `020` | 요청 제한을 초과하였습니다. 일반적으로는 20,000건 이상의 요청에 대하여 이 에러 메시지가 발생되나, 요청 제한이 다르게 설정된 경우에는 이에 준하여 발생됩니다. |
| `021` | 조회 가능한 회사 개수가 초과하였습니다.(최대 100건) |
| `100` | 필드의 부적절한 값입니다. 필드 설명에 없는 값을 사용한 경우에 발생하는 메시지입니다. |
| `101` | 부적절한 접근입니다. |
| `800` | 시스템 점검으로 인한 서비스가 중지 중입니다. |
| `900` | 정의되지 않은 오류가 발생하였습니다. |
| `901` | 사용자 계정의 개인정보 보유기간이 만료되어 사용할 수 없는 키입니다. 관리자 이메일(opendart@fss.or.kr)로 문의하시기 바랍니다. |

### 공통 요청 인자

모든 DS002 API는 동일한 4개의 필수 요청 인자를 사용합니다.

| 키 | 명칭 | 타입 | 필수 | 설명 |
|----|------|------|------|------|
| `crtfc_key` | API 인증키 | STRING(40) | Y | 발급받은 인증키(40자리) |
| `corp_code` | 고유번호 | STRING(8) | Y | 공시대상회사의 고유번호(8자리) |
| `bsns_year` | 사업연도 | STRING(4) | Y | 사업연도(4자리) ※ 2015년 이후 부터 정보제공 |
| `reprt_code` | 보고서 코드 | STRING(5) | Y | 1분기보고서: 11013 / 반기보고서: 11012 / 3분기보고서: 11014 / 사업보고서: 11011 |

- 메서드: GET | 인코딩: UTF-8 | 출력포멧: JSON (.json) / XML (.xml)

### 공통 응답 필드

모든 DS002 API 응답에 포함되는 공통 필드입니다.

| 키 | 명칭 | 설명 |
|----|------|------|
| `result > status` | 에러 및 정보 코드 | 공통 에러코드 참조 |
| `result > message` | 에러 및 정보 메시지 | 공통 에러코드 참조 |
| `list > rcept_no` | 접수번호 | 접수번호(14자리) ※ 공시뷰어: `https://dart.fss.or.kr/dsaf001/main.do?rcpNo=접수번호` |
| `list > corp_cls` | 법인구분 | Y(유가), K(코스닥), N(코넥스), E(기타) |
| `list > corp_code` | 고유번호 | 공시대상회사의 고유번호(8자리) |
| `list > corp_name` | 법인명 | 회사명 |
