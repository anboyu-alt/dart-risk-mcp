종목을 입력하면 최근 공시를 훑어 신호를 감지합니다. 사실 관찰이며 판정이 아닙니다.

▍WHAT IS THIS — 이 페이지가 하는 일

한국 상장사의 이름을 입력하면, 금융감독원 전자공시(DART)에서 그 회사의 최근 공시를 그 자리에서 실시간으로 수집해 불공정거래와 상관이 높은 위험 신호(횡령·거래정지·무상감자·CB 반복 발행 등 51종)를 타임라인과 해설로 보여줍니다.

  • 조회에는 본인의 무료 DART API 키가 필요합니다 — 키는 이 브라우저에만 저장되고, 서버에 아무것도 남지 않습니다
  • 점수·등급·매수 추천은 없습니다 — 공시라는 공개 기록의 사실 표기와 원문 링크만 제공합니다
  • 이 페이지는 오픈소스 분석 엔진 dart-risk-mcp의 웹 축약판입니다 — AI와의 대화(MCP)에서는 원문 열람·자금 추적까지 더 깊이 들어갑니다
▍FIRST RUN — API KEY

이 도구는 서버에 데이터를 쌓지 않습니다. 입력한 키는 이 브라우저에만 저장되고, 공개 코드로 검증 가능한 무저장 릴레이를 거쳐 금융감독원 DART로만 전송됩니다. 키가 없다면 opendart.fss.or.kr에서 이메일 인증으로 2분 만에 무료 발급됩니다(일 20,000건).
