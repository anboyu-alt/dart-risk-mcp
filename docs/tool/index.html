저장된 키:

임원 겸직 비교 2~5개사 · 최근 4개 사업연도 임원현황 교집합

문제 세력은 회사를 인수할 때마다 투자조합 이름을 바꾸지만, 등기임원 이름은 공시에 남는 고정점입니다. 비교할 회사들을 추가하면 DART 임원현황을 겹쳐 2개사 이상에 등장하는 인물을 찾습니다. (MCP find_actor_overlap의 임원 차원과 동일 로직)

본 도구의 모든 표시는 DART 공시라는 공개 기록의 사실 표기이며, 특정 기업에 대한 판정·점수·등급·투자 권유가 아닙니다. 신호는 "원문을 확인해 볼 지점"의 표시일 뿐입니다 — 각 항목을 클릭해 반드시 DART 원문을 확인하세요. 조회는 사용자의 API 키로 사용자가 시작하며, 이 페이지는 어떤 데이터도 서버에 저장하지 않습니다. · dart-risk-mcp (MIT)
