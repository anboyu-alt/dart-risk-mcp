```
케이비증권 주식회사(본건펀드1의 신탁업자 지위에서)     800,000,000
미래에셋증권 주식회사(본건펀드2의 신탁업자 지위에서)  4,500,000,000

미래에셋증권 주식회사(본건펀드4의 신탁업자 지위에서)  1,000,000,000
미래에셋증권 주식회사(본건펀드5의 신탁업자 지위에서)  1,000,000,000
에스더블유 아이 신기술조합                             2,200,000,000
```

**핵심 보정:**
- `fetch_cb_issue_decision` / `fetch_bw_issue_decision` / `fetch_eb_issue_decision` 파라미터 `rcept_no` → `corp_code+bgn_de+end_de` 수정.
- 구조화 엔드포인트(`cvbdIsDecsn`)는 deal term만 반환하고 `actnmn` 필드가 없음 → HTML `<TE ACODE="ISSU_NM">` 테이블 파싱으로 보완.
- `_fetch_text` 20,000자 상한 제거 (CB 섹션이 char 23,102~에 등장하는 샘플 존재).

**HTML 파싱 접근 방식:**
- DART 표준 공시는 `<TE ACODE="X">content</TE>` 구조 — ACODE 속성으로 컬럼을 식별.
- CB: `ACODE="ISSU_NM"`(대상자명), `ACODE="ISSU_AMT"`(금액).
- Rights: `ACODE="PART"`(제3자배정 대상자), `ACODE="ALL_CNT"`(배정주식수).
- 이전 heuristic(토큰 길이·헤더 문구 필터)은 "선정경위" 프로즈 텍스트를 인수자로 오인했음 — ACODE 기반 파싱으로 false positive 제거.
